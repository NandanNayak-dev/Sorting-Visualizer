# 📊 Sorting Visualizer

A web-based Sorting Visualizer built with **HTML, CSS, and Vanilla JavaScript** that helps you understand how different sorting algorithms work through animations.

## 🚀 Live Demo
**[Sorting Visualizer](https://nandannayak-dev.github.io/Sorting-Visualizer/)**

## 🎨 Features
- **Visualize Multiple Algorithms:**
  - 🫧 **Bubble Sort**: Compares adjacent elements and swaps them if they are in the wrong order.
  - 🎯 **Selection Sort**: Repeatedly selects the minimum element and places it at the sorted end.
  - 📥 **Insertion Sort**: Builds the final sorted array one item at a time by finding its correct place.
  - 🔀 **Merge Sort**: A divide and conquer algorithm that splits the array in halves, sorts them, and merges them.
- **Customizable Controls:**
  - Change the **Array Size** (number of bars).
  - Adjust the **Sorting Speed** to watch algorithms step-by-step or zoom through them.
  - **Generate** a new random array on the fly.
- **Responsive & Clean UI:** A visually pleasing dark-themed interface with smooth color-coded animations (e.g., comparing, swapping, and sorted states).

## 🛠️ Tech Stack
- **HTML5**: Semantic web structure.
- **CSS3**: Responsive flexbox layout, UI styling, and aesthetic dark mode.
- **JavaScript (Vanilla)**: DOM manipulation, algorithm logic, asynchronous functions (`async/await`) for managing animation delays.

## 📂 Run Locally

Want to run this yourself? It's super simple!

1. Clone the repository:
   ```bash
   git clone https://github.com/NandanNayak-dev/Sorting-Visualizer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Sorting-Visualizer
   ```
3. Open the `index.html` file in your favorite web browser. 
   *(No server, npm, or extra dependencies needed!)*
